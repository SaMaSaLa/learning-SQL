# learning-SQL

### Login

```
mysql -u<username> -p
```

### See what are all the databases available

```
show databases;
```

### Create new database

```
create database todos;
```

### Start using a specific database

```
use todos;
```

### Show all tables inside current database

```
show tables;
```

### Delete table

```
drop table todos;
```

### Delete entire database

```
drop database todos;
```
